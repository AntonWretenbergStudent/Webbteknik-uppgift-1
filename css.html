Vad är en regel i CSS och vad består den av?

En regel i css består av två delar och dom två delarna är Selektor och Deklarationsblock. Och det selektorer gör är att dom bestämmer vilket html element som ska regeln ska appliceeras på.

Och vad ett Deklarationsblock gör är att den bestämmer vad som ska göras som tillexempel vad för färg elementet ska få och storlek.

Vad är skillnaden mellan id och class? När bör man använda id respektive class?

Id används för att identifiera specifika element som tillexempel css eller javascript

Men class kan användas för att gruppera flera element som har samma specifika element
